# 🧠 Fit AI — Back-end (API)

API da plataforma **Fit AI**, desenvolvida durante o **Bootcamp Gestão de Treinos** com o professor **Felipe Rocha**.

Responsável por gerenciar regras de negócio, autenticação, geração de treinos e integração com serviços de inteligência artificial.

---

## 🚀 Funcionalidades

- Geração de treinos personalizados com IA
- Gerenciamento de usuários e perfis
- Controle de consistência (streaks)
- Autenticação via Google
- Integração com serviços de IA

---

## 🧩 Tecnologias utilizadas

- Node.js
- Fastify
- PostgreSQL
- Prisma ORM

  ## 📦 Bibliotecas utilizadas

- Better Auth
- Vercel AI SDK
